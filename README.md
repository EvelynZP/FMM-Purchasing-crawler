# FMM-Purchasing-crawler

该项目为每日早8点自动爬取新发地官网发布的当日菜价，制作可视化文档，自动邮件发送给相关部门人员

![封面图片](https://github.com/Tingjbhappy/FMM-Purchasing-crawler/blob/master/%E5%B0%81%E9%9D%A2%E5%9B%BE%E7%89%87.png)
## 目录
- 爬虫
  - 定制请求
  - 解析返回数据
  - 存储数据
- 数据可视化
  - 每日采购数量条形图
  - 分类占比饼图
  - 价格波动折线图
- 自动邮件
